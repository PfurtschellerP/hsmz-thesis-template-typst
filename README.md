# Unofficial Thesis Template for the Hochschule Mainz (HSMZ)

A [Typst](https://typst.app/) template for thesis writing at the Hochschule Mainz (HSMZ). This template provides a basic structure and adheres to the [guidelines set by the university](https://www.hs-mainz.de/fileadmin/Wirtschaft/Services/Pruefungsamt/pdf/Downloads/leitfaden/Leitfaden_wissenschaftlicher_Arbeiten_Deutsch.pdf).

## Collaboration

This project is open for collaboration. If you have suggestions, improvements, or would like to contribute, please feel free to open an issue or submit a pull request.
